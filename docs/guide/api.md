# 系统相关接口
## 1、用户登录
### 1.1接口描述

用户登录

### 1.2 请求方式

POST  /ul/login

### 1.3 输入参数

| 参数名称  | 必选 | 类型 | 描述       |
| --------|-----|------|-----------|
| username | 是 | string | 用户名 |
| password | 是 | string | 用户密码|

### 1.4 输出参数
| 参数名称    | 类型 | 描述      |
|---------|------|---------|
| code    | string | http状态码 |
| message | string | 登录提示信息  |
| token   | string | 登录给的签证信息，有效期为24小时 |

**输入示例**
